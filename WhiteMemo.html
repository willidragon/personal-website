Overview

White Memo takes the user into the head of Ling, the protagonist who's trying to solve a mystery through looking and interactive with objects.

In order to achieve this effect, my project parter and I delved into the world of Mixed Reality- one that combines both Virtual and Augmented Reality. This allows the user to manipulate both hands freely while achieving an immersive and surreal effect. We wanted to use this project to explore the boundaries of story telling technique.

Users


Setup

This is the setup where users can experience our app. The various props are distributed around the table for users to interact with using AR. Not all of the objects are crucial to the story, many we placed are there to increase the difficulty and to enhance the content for the players.



Signup

In order for the users to begin the experience, they must first signup. The required steps includes taking a picture as we implemented a facial blending feature which blends the user's face into many photos using AR.



Signing In

After signing up, when you open the app, it'll prompt you to login.


Features




Interaction

The reticle shows up and displays a loading indicator, once the hourglass empties, the user's interaction with the feature is logged- and triggering responses will ensue. In this case, the player will hear the narrator's inner thoughts.



Facial Blend

We implemented a feature in which the user's face is mapped onto pictures of story characters during their experience- its as if they are part of the story. When the user takes a photo during signup, their photos are kept in the cloud and later pulled once they begin the game.



Switching between AR/VR

As the story progresses, players will sometimes reach waypoints which will take them to VR scenes which has other story sequences.

Real Time Object Interaction

The real fun begins here, when user can interact with objects in real time!
